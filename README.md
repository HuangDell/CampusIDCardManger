## 校园卡类管理系统
* **
### 开发环境：  
**Visual Studio 2019 && MySQL 8.0**  
### 关于数据库：  
**数据库的相关信息在bin/config.json文档中记录,可能会需要修改IP地址**
### 关于软件：  
* **校园卡每个学号最多注册一张**  
* **储蓄卡每个学号最多注册三张**  
* **每张校园最多绑定三张储蓄卡**
* **校园网可能会连不上数据库，推荐使用非校园网**
* **如果显示无法连接服务器，请联系我**
